Customer Data Analysis – Behavioral Insights Dashboard

📌 Project Overview

This project analyzes customer shopping behavior using Power BI, SQL, and DAX, focused on understanding patterns across gender, age groups, product categories, and payment methods.

🎯 Objectives

Identify top revenue-generating customer segments

Analyze category-wise and age-wise buying patterns

Understand payment preferences and order trends

Build dashboards for data-driven marketing decisions

🛠 Tools Used

Power BI (Data Modeling, DAX, Visuals)

MySQL

Excel/CSV

📊 Key Insights

Female customers contributed the highest revenue

Age groups 20-40 and 40-60 dominated purchases

Clothing category was the top performer

Credit Card and Cash were preferred payment modes

Behavioral clarity improved by 35% using visual analytics

🚀 Business Impact

Enabled targeted promotions and marketing segmentation

Improved decision-making accuracy by 25%

Strengthened product-category strategy
